# SmartAffiliate - AI Reviews Hub 2026

This is an AI-powered affiliate review site built with **Jekyll** and deployed via **GitHub Actions**.

## 🚀 Deployment Status
The site is automatically built using Docker and Jekyll via the `.github/workflows/jekyll-ci.yml` file.

## 📁 Key Pages
- [Home](index.html)
- [How to Use AI for SEO](how-to-use-ai-for-seo.html)
- [ChatGPT Review](chatgpt-review.html)

## 🛠️ Technical Setup
- **Theme:** Custom CSS
- **Engine:** Jekyll 4.x
- **CI/CD:** GitHub Actions (Dockerized Builder)
